Medical Appointment Scheduler (C++)
Project Description
The Medical Appointment Scheduler is a C++ console-based application designed to manage patient appointments in a clinic or hospital. It organizes patients efficiently and schedules appointments using a queue system (First Come First Serve).
Features
Add new patient details (ID, Name, Age, Disease)
Schedule appointments in order
View patient list
Manage waiting queue
Simple and user-friendly console interface
Technologies Used
C++
Object-Oriented Programming (OOP)
Data Structures (Queue, Vector)
File Handling (optional)
Working Principle
Patients are added to a queue as they register. The system processes appointments in FIFO (First In First Out) order, ensuring fair scheduling and reducing waiting time confusion.
Objective
To simulate a real-world hospital appointment system and improve efficiency in managing patient flow using programming concepts.
